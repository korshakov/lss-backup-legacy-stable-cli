## Install
Make sure you are downloading the latest release.
```
tar -xvf v1.0.0.tar.gz
rm v1.0.0.tar.gz
mv lss-backup-v1.0.0 lss-backup
cd lss-backup
chmod +x *.sh
chmod +x functions/*.sh prep-dependencies/*.sh.prep
bash install-lss-backup.sh
```
